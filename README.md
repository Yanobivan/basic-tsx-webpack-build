# Basic-tsx-webpack-build

## Plugins and features:

- stylelint (stylelint-config-standard): BEM naming convention & Material-ui
- huski
- tsx
- babel
- babel-plugin-transform-imports
- scss
- postCSS
- eslint
- prettier
- lint-staged
- normalize.css

## Installation

```sh
npm i
```

## How to run?

**Dev version:**

```sh
npm start
```

**Prod version:**

```sh
npm build
```

## How to fix?

**Show mistakes**

```sh
npm run stylelint
```

**Fix mistakes**

```sh
stylelint:fix
```
